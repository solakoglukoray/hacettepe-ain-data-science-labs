# AIN214 PA1 — Flight Delays EDA + SQL Analysis

> AIN214 Data Science Lab · Fall 2025

## Overview

Two-part assignment: exploratory data analysis on US flight delays, and SQL-based analysis on a food delivery dataset.

## Part 1 — Flight Delays Dataset

US Department of Transportation flight data.

- Load and inspect multi-file CSV dataset
- Statistical exploration and analysis
- Visualization of delays, cancellations, and trends

## Part 2 — Foodpanda SQL Analysis

Relational database built from a food delivery CSV using `sqlite3`.

**Queries include:**
- Top 3 restaurants by total revenue
- Total quantity sold by category per restaurant
- Top 5 customers by spending
- Average restaurant ratings
- Order status analysis

## Run

```bash
jupyter notebook ain214_pa1.ipynb
```

## Topics

`Pandas` `SQLite3` `EDA` `Data visualization` `SQL joins & aggregations`
